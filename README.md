# moonlighter.ml
Web hosting for [moonlighter.ml](http://moonlighter.ml/)

## A little help
`CNAME` file in the repository stores custom domain name to be used. `A records` of the custom domain should be set to: 192.30.252.153 and 192.30.252.154.
